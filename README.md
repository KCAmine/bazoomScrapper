## This google extension scrapper was developped only for a certain website wich might not work if  you decide to use it for a random website and your personal use, still tho the code is open source and here are the steps to install the extension: 


1. Download the zip file.
2. Unzip it
3. In chrome search bar tap : chrome://extensions
4. Look for Load unpacked extension or Look unpacked
5. Select the folder
6. You should now be able to use it as an extension but like said above , it might not work since it was developped for a special case on a certain mission.
